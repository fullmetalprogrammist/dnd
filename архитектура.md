# Описание приложения

- Приложение по изучению английского языка по репликам из фильмов
- Пользователь выбирает фильм и ему открывается страница с карточками
  - В каждой карточке находится кадр из фильма и несколько реплик из соответствующего момента фильма



# Создание нового проекта

- В компоненте CreateNewProject создания нового проекта у нас есть
  - Поле ввода под код проекта
    - Когда пользователь ввел код, мы сразу проверяем его доступность через запрос на сервер
  - Поле ввода для выбора файла с репликами
- Когда код проекта не занят и выбран файл с репликами, можно создать проект
- При этом в БД создается запись под проект, реплики загружаются в БД
- После этого пользователь попадает в компонент редактирования проекта EditRoom



- Варианты оформления CreateNewProject 
  - 1 вариант: отдельная страница под диалог создания
    - тут поле ввода кода проекта и поле загрузки файла с репликами
    - проверяем код, потом сохраняем проект, реплики грузятся в БД и дальше переход на страницу редактирования
    - оказываемся где-то вроде editroom/dnd
  - 2 вариант: модальное окно под диалог создания
    - поле ввода кода проекта и загрузки файла находятся в модальном окне
    - когда все ок, оно просто закрывается и оказываемся на странице редактирования
